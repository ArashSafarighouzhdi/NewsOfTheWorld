# 📰 News Of The World

A modern news web application that provides users with the latest headlines from around the world in a clean and user-friendly interface.

Live Demo: https://news-of-the-world-xi.vercel.app/

---

## 🚀 Features

- 🌍 Browse latest global news
- 🗂️ Categorized news (e.g., Technology, Sports, Business)
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React, JavaScript
- **Styling:** CSS Module
